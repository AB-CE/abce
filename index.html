Welcome to ABCE’s documentation!¶

ABCE is a Python Agent-Based Complete Economy Platform, written by Davoud Taghawi-Nejad. The walk through, explains you how to set up a simulation. In the agent model you learn how to set up an agent and how to trade with other agents. The households and firms classes allow you to produce with different production functions and consume with according production functions. In order to get result you need to use the logging and panel data creation tools described in Obserning Agents.
